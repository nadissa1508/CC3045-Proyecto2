# CCC3045-Proyecto2

Proyecto 2 del curso Inteligencia Artificial

## Módulos

- **CSP**: Asignación de variedades de café a parcelas
- **Bayesian Network**: Red bayesiana para riesgo de enfermedades (roya)
- **ML**: Predicción de rendimiento con KNN, DecisionTree y Redes Neuronales
- **MDP**: Proceso de decisión de Markov para optimización estacional
- **Dashboard**: Interfaz interactiva para visualización y recomendaciones

## Instalación

```bash
pip install -r requirements.txt
```

Ver notebooks en `notebooks/` para tutoriales detallados.
